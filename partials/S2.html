Notes on the table:
  1. The subscripts near atoms are SMARTS descriptors. A full description is available from Daylight but some of the most common in this list are shown below
    • R - Atom in specified number of rings eg: R0 is in no rings
    • r - Smallest ring atom is in contains specified number of bonds eg: r5 is in 5 member ring
    • a - Atom is aromatic
    • H = Atom connected to exactly the specified number of hydrogen atoms
    • D - Atom has specified number of non hydrogen bonds
    • -;@ - The dotted bond is a single bond but not in a ring
    • $() - Atom is connected to the group in parenthesis. Used to specify multiple electron withdrawing groups
