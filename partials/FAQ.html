Frequently Asked Questions

How do I use this website?

There are many ways you can use this website but we have developed a few tours that show the basics.

  • General Website Tour
  • Metabolomics Tour

How do you generate in silico metabolites?

These compounds are generated from automatically generated reaction rules from known biochemistry on MetaCyc. For more information please refer to the original MINE publication.

How can I integrate your database into my tool/pipeline?

We offer a REST API in python which covers all the functionality of this website (and more). These clients are available at GitHub. For example, you could use the REST API to download all MINE compounds.

I found an error/I'm skeptical about the validity of a compound

We would love to hear about it. Please let us know by using our bug reporting form. If reporting a bug, please include information as to how the error was generated.

Are there any restrictions on the use of this website and APIs?

MINEs are provided as a freely available resource. There are no licensing restrictions for non-commercial use but users are requested to cite the following publications if their work utilizes the MINE resources:
