Frequently Asked Questions

There are many ways you can use this website but we have developed a few tours that show the basics.

  • General Website Tour
  • Metabolomics Tour

How do you generate in silico metabolites?

These compounds are generated from hand-curated reaction rules generalized form chemistry according to the Enzyme Commission classification system. For more information please refer to the original MINE publication

How can I integrate your database into my tool/pipeline?

We offer API clients in JavaScript, Python and Perl which cover all the functionality of this website. These clients are available at GitHub. You may also download SDF files of the generated metabolites in the database. Due to licencing restrictions, we are not able to redistribute the original database compounds (i.e. KEGG COMPOUND) but these compounds are available from the original sources.

I found an error/ I'm skeptical about the validity of a compound

We would love to hear about it. Please let us know by using our bug reporting form. If reporting a bug, please include information as to how the error was generated.

Are there any restrictions on the use of this website and APIs?
